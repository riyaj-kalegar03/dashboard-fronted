# Debt Recovery Dashboard - Auth Module

This scaffold implements the authentication module (Phase 1) using React + Vite, Tailwind CSS, and Axios with JWT interceptors.

Setup:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Notes:
- API base URL can be set via `VITE_API_BASE` environment variable.
- Routes: `/login`, `/dashboard`.
